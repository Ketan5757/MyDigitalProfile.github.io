Project Overview

This project utilizes deep learning models trained with Google AutoML to classify images of cats and dogs. The system leverages convolutional neural networks (CNNs) to achieve high accuracy in image recognition.

Key Features

  • Automated image classification using deep learning
  • High-accuracy model trained with Google AutoML
  • Supports multiple image categories
  • Optimized for real-time predictions
  • User-friendly interface for image uploads

Technologies Used

  • Machine Learning: Google AutoML, TensorFlow
  • Frontend: React.js, Tailwind CSS
  • Backend: Node.js, Express.js
  • Database: Firebase
  • Cloud Services: Google Cloud Platform (GCP)
