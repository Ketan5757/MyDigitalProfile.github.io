Hotel Management System

Project Overview

I have developed a Hotel Management System that efficiently manages room bookings, customer check-ins, billing, and administrative operations. The system ensures seamless and automated hotel operations with an interactive interface for both customers and admins.

The platform integrates secure payment processing, real-time room availability tracking, and an admin dashboard that allows hotel staff to manage reservations, update room statuses, and process customer check-outs easily.

Key Features

  • Automated room booking & real-time availability tracking
  • Secure customer check-in and check-out process
  • Integrated billing system with Stripe for secure payments
  • Admin dashboard to manage reservations and customer records
  • Optimized room allocation and guest data management
  • User-friendly interface with real-time database updates

Technologies Used

  • Frontend: Java Swing
  • Backend: Java, JDBC
  • Database: MySQL (Relational Database for room & customer management)
  • Payment Integration: Stripe API
