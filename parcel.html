Parcel Service Application

Project Overview

I have developed a Parcel Service Application that enables users to manage and track shipments efficiently. It provides real-time tracking, automated notifications, and seamless user interaction.

Key Features

  • Real-time parcel tracking
  • Automated delivery notifications
  • Secure user authentication with Clerk.js
  • Intuitive and user-friendly interface
  • Integration with multiple courier services

Technologies Used

  • Frontend: React.js, Tailwind CSS
  • Backend: Node.js, Express.js
  • Authentication: Clerk.js
  • Database: MongoDB Atlas
  • API Integrations: Courier APIs for real-time tracking
